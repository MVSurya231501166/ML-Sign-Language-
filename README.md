# ML-Sign-Language-
A real-time Sign Language Recognition system built using Python, OpenCV, and MediaPipe. The application detects hand landmarks from live webcam input, extracts coordinate-based features, and classifies gestures using a trained ML model.
Features:

      Real-time gesture detection
      
      Landmark-based feature extraction
      
      Custom dataset training
      
      High-speed prediction
      
      Modular ML pipeline
      
      Easy extensibility for adding new signs

Model Training:

      Dataset collected manually using webcam
      
      Landmark coordinate preprocessing and normalization
      
      Supervised classifier trained for multi-class gesture recognition
      
      Accuracy evaluation and confusion matrix analysis
